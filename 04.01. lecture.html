Module 4: Industry Standards

Welcome to Compliance Assessments

Industry standards form the backbone of cybersecurity governance. They provide structured frameworks to protect sensitive information, manage risks, and demonstrate compliance.

Today's Focus:

  • NIST CSF: The Cybersecurity Framework (Identify, Protect, Detect, Respond, Recover).
  • ISO 27001: Information Security Management Systems (Plan-Do-Check-Act).
  • PCI-DSS: Payment Card Industry Data Security Standard.
  • Mapping: Translating legal requirements (GDPR/HIPAA) into technical controls.

Code Demonstration: Automating 'Identify'

Task: Asset Discovery Simulation

This Python script simulates the IDENTIFY function by scanning a subnet for active assets and basic OS fingerprinting. In a real scenario, this helps build an asset inventory.

asset_discovery.py
import nmap # Import network mapper library used for discovery import json # Import JSON library to format the output report   def identify_assets(subnet): # Initialize the scanner object to perform network operations nm = nmap.PortScanner() # Scan the specific subnet for hosts using Ping Scan arguments # '-sn' disables port scan (ping scan only) for speed print(f"[*] Starting scan on {subnet}...") nm.scan(subnet, arguments='-sn')   assets = [] # Iterate through all hosts found to be 'up' for host in nm.all_hosts(): # Create a dictionary for each asset with critical details asset = { "ip": host, "status": nm[host].state(), "hostname": nm[host].hostname() } assets.append(asset) print(f" [+] Found: {host} ({asset['hostname']})")   return assets   # Main execution block representing the Framework Profile implementation if __name__ == "__main__": inventory = identify_assets('192.168.1.0/24') print("\n[*] Asset Inventory Generation Complete.")

Code Demonstration: Detecting Unencrypted Data

Task: PAN Scanning (Data Discovery)

This Bash script simulates searching for unencrypted Primary Account Numbers (PAN) in a data directory using regex. This maps to PCI-DSS Requirement 3 (Protect stored cardholder data).

pan_scan.sh
#!/bin/bash # Define the regex pattern for a Credit Card (Simple 16-digit check) PATTERN="[0-9]\{16\}" TARGET_DIR="./data_storage"   echo "[*] Starting Compliance Scan for Plaintext PANs..."   # 'grep' searches for text patterns inside files recursively (-r) # '-n' outputs line numbers where matches are found grep -r -n "$PATTERN" $TARGET_DIR | while read -r line ; do # Parse the output to get filename and context echo " [!] VIOLATION FOUND: Unencrypted CC data detected!" echo " Details: $line" done   echo "[*] Scan complete. Check findings report."

Lesson 3: Mapping Frameworks

Translation

Cybersecurity professionals often need to map legal requirements (like GDPR or HIPAA) to technical controls (like NIST). This process ensures that abstract laws are met with concrete security configurations.

Legal Requirement (Source) Technical Control (NIST CSF) Implementation (Action)
GDPR Art. 32
Security of Processing		 PR.DS-1: Data-at-rest is protected. Enable AES-256 encryption on all databases.
HIPAA §164.312
Audit Controls		 DE.CM-1: The network is monitored. Configure SIEM log forwarding.
PCI-DSS Req 8
Identity Access		 PR.AC-7: Users are authenticated. Enforce MFA and 12-char passwords.

Code Demonstration: Auditing Compliance

Task: Password Policy Audit

This PowerShell script queries the Active Directory password policy to ensure it meets the mapped control requirements (e.g., NIST PR.AC-1 or PCI Req 8).

audit_policy.ps1
# Retrieve the default domain password policy $policy = Get-ADDefaultDomainPasswordPolicy   Write-Host "[*] Auditing Domain Password Policy against NIST Standards..."   # Check minimum password length (Standard requires > 8 or 12 depending on level) if ($policy.MinPasswordLength -lt 12) { Write-Warning " [FAIL] Length is $($policy.MinPasswordLength). Req: 12" } else { Write-Host " [PASS] Length meets requirement." -ForegroundColor Green }   # Check complexity requirements if ($policy.ComplexityEnabled -eq $true) { Write-Host " [PASS] Complexity is enabled." -ForegroundColor Green }
